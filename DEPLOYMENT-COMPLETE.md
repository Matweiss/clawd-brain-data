## What Each Agent Does Now

### CLAWD PRIME
1. **Receives** every request from Mat, Clawd Prime, or other agents
2. **Interprets** intent (what's Mat really trying to achieve?)
3. **Routes** to best-fit agent with clear brief
4. **Supervises** with specific triggers (24-48h stalls, blockers, conflicts)
5. **Learns** from Mat's overrides and adjusts future decisions
6. **Escalates** high-impact decisions that need human judgment

### WORK AGENT
1. **Maintains** dashboard as shared command center
2. **Synthesizes** context from HubSpot + Gmail + Avoma
3. **Detects** at-risk deals (stage-aware, importance-weighted)
4. **Drafts** emails in Mat's voice (learning from every edit)
5. **Presents** ready-to-act summaries with proposed next steps
6. **Learns** from Mat's approvals/rewrites and refines tone model

### BUILD AGENT
1. **Takes** from Clawd Prime's prioritized backlog
2. **Ships** multiple times per week (frequent, safe, small)
3. **Owns** MVP vs. polish decision based on stakes
4. **Responds** to P0 issues immediately (drop everything)
5. **Debugs** root causes and escalates scope/trade-off decisions
6. **Proposes** improvements when spotting repeated manual work

### RESEARCH AGENT
1. **Responds** to research requests from any agent
2. **Proactively** scans before important meetings
3. **Gathers** in hierarchy: snapshot → people → signals → angles
4. **Presents** synthesis not dumps (with talking points)
5. **Cites** all facts and separates inference from strategy
6. **Learns** which insights actually move deals

### LIFESTYLE AGENT
1. **Suggests** fun activities daily (proactive)
2. **Guards** yoga blocks (hard protected, non-negotiable)
3. **Protects** creative time (soft protected, can move)
4. **Catches** "about to skip" and nudges gently
5. **Manages** stress with resets (Diggy, yoga, break)
6. **Celebrates** wins actively with affirmation & praise
