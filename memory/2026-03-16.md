### CorePower Login Issue
- **Problem:** User couldn't log in to CorePower on Tandem
- **Root Cause:** Tandem uses isolated cookie/storage (`persist:tandem` partition)
- **Chrome cookies/sessions NOT shared** with Tandem
- **Solution:** Must manually log in on Tandem (different session storage)

### Key Lessons
- **CDP browser ≠ visible Chrome** — user sees normal Chrome, CDP runs headless/background
- **Tandem cookies isolated** — can't use Chrome login credentials
- **Tandem needs SSH reverse tunnel** (-R flag) to forward port 8765 from Mac to VPS
- **API token on Mac** — cannot read from VPS; must be shared manually
