### Escalation Flow
```
Request → Vandalay (Review & Score) → Sloan (Package) → Bob/Agent (Build)
              ↓
    API integrations → Clawd → Mat (for approval)
```

### Key Rules
- **Self-execute:** Routine maintenance, bug fixes, scheduled tasks, known patterns
- **Escalate to Vandalay:** New features, workflow changes, new skills, strategic decisions, cross-agent coordination, unclear/high-risk work
- **Escalate to Sloan:** After Vandalay approves, when execution plan needs clarification
- **Escalate to Clawd:** Any API integration (always → Mat)
