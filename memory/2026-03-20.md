
## Schema Guardrail (Added 2026-03-20)
For all future tool/extension schema authoring (especially OpenAI-style tools), enforce:

1. Every parameter schema must be valid JSON Schema.
2. Any property with `"type": "array"` must include `"items"` with explicit element schema.
3. Never emit arrays without `items`.
4. Prefer explicit `required` and primitive types.
5. Preflight self-check before finalize:
   - array missing `items`
   - object/property type definition gaps
   - `required` entries pointing to non-existent properties

Hard rule: reject output if any schema contains `"type":"array"` without `"items": {...}`.
