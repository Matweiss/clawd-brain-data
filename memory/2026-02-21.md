## Browser Relay Root Cause Found (9:31 AM PST)

**Mac Config Issue:**
```json

  "url": "ws://31.97.142.214:18789"  // PLAINTEXT to public IP - BLOCKED

```

**Security Error:**
OpenClaw blocks plaintext WebSocket (`ws://`) to non-loopback addresses - prevents credential/data exposure

**Options to Fix:**
1. **wss:// (secure)** - Requires SSL cert on VPS (complex)
