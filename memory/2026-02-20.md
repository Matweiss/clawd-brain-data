**Key Features Designed:**

1. **Persistent Sidebar Chat**
   - Chat with Clawd on every dashboard page
   - Assign tasks directly: "Draft follow-up for Acme" → I execute
   - Task states: assigned → working → awaiting_review → done

2. **Meeting Battle Cards**
   - Auto-generated prep 45 min before meetings
   - Sources: HubSpot + Avoma + LinkedIn + ZoomInfo
   - Company info, attendee research, last meeting notes, suggested talking points

3. **Intelligent Task Management**
   - Suggested tasks (auto-generated based on triggers)
   - Drag-and-drop reordering
   - Urgent section
   - "Send to Clawd" button → I execute, return for review
   - Undo completed tasks

4. **Stale Deals View**
   - Dedicated page for deals needing attention
   - Auto-creates tasks for each stale deal
   - Recommended actions based on stage thresholds

5. **Enhanced Pipeline Table**
   - Contact info (email, phone)
   - Sortable columns
   - Filters (stage, status, last activity)

6. **Clickable KPI Cards**
   - Click "Pipeline" → Pipeline tab
   - Click "Stale Deals" → Stale Deals view

**Data Integration Required:**
- HubSpot API (deals, contacts, companies)
- Avoma API (meeting transcripts)
- Google Calendar (meetings)
- Gmail (draft creation)
- LinkedIn (via browser scraping)
- ZoomInfo (company/contact enrichment)

**Implementation Approach Proposed:**
- **Option A:** Embedded chat in dashboard (spec design)
- **Option B:** Telegram chat → I update dashboard (faster to build)

**Decision:** Leaning toward Option B initially, add embedded chat later
