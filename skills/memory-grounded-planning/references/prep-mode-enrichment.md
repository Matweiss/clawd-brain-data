# Prep-Mode Enrichment

When the user asks how to prepare, what to pack, what to bring, or what they may be forgetting:

## Workflow
1. Recall the likely event/trip/context from memory.
2. Confirm if ambiguous.
3. Gather missing high-value context when useful:
   - weather
   - destination norms
   - duration
   - event type
   - known dinners / meetings / activities
4. Answer with a context-specific prep plan.

## Good output shape
- likely event identified
- quick confirmation if needed
- categorized checklist
- specific clothing/logistics guidance
- reminders tied to known plans
- "don't forget" items driven by memory, not generic templates

## Examples of memory-driven reminders
- golf trip -> clubs / rental decision / golf clothes
- NYC dinner -> one nicer dinner outfit
- late-night flight -> plane outfit, charger, sleep plan
- shared trip -> coordinate packing or hotel logistics with Sarah if relevant
