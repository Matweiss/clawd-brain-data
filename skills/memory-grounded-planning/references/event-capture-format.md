# Event Capture Format

Use this format when logging future commitments that may matter for planning, prep, reminders, packing, or schedule recall.

## Canonical target
- `memory/upcoming-events.md`

## What belongs here
- flights
- trips
- dinners
- reservations
- meetings with fixed dates
- appointments
- weddings / parties / social events
- key work events that affect real-world planning

## Entry pattern
- **YYYY-MM-DD or YYYY-MM-DD to YYYY-MM-DD — Event name**
  - Time / outbound / return / reservation details
  - Location / route / venue
  - Companions if relevant
  - Notes: special logistics, dress needs, goals, reminders

## Logging rule
When Mat mentions a future dated plan that is likely to matter later, capture it in `memory/upcoming-events.md` and, if appropriate, also mention it in the day's memory note.

## Recall rule
For schedule, packing, prep, and "what do I have coming up" questions, search `memory/upcoming-events.md` early.
