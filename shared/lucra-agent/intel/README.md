# Intel Folder

## New asset from Scout
- `prospect-intel-card-template.md` — 5 minute standardized company intel card for Lucra prospecting.

## Recommended workflow
1. Start with company name or domain.
2. Fill the template in this folder as `YYYY-MM-DD-company-name.md`.
3. Keep sources inline and end with a clear Scout verdict.
4. Prioritize contacts by who owns growth, product, partnerships, or fan engagement.
5. Hand Luke the finished card plus one recommended outreach angle.

## Suggested filename pattern
- `2026-04-11-underdog.md`
- `2026-04-11-draftkings.md`
- `2026-04-11-sleeper.md`
