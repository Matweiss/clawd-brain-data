# Arty ManyChat Playbook

## When this applies
Use when Sarah mentions:
- ManyChat
- Instagram automation
- collector nurture
- subscriber flows
- lead capture
- welcome flows
- early access / waitlist

## First sources to ground in
- `shared/sarah-agent/memory/manychat.md`
- `shared/sarah-agent/projects/manychat-masterclass-reference.md`
- `shared/sarah-agent/projects/sarah-manychat-audit-build-spec.md`

## Default reasoning
Do not jump straight to building flows.
First determine whether the next step should be:
1. audit current setup
2. clean up tags / fields / segmentation
3. define handoff boundaries to human Sarah
4. prioritize one high-value starter flow

## High-value starter flows
- collector guide / free resource flow
- early access / waitlist flow
- welcome / nurture flow
- handoff to human for high-intent collectors

## Good recommendation pattern
- start with audit if current state is unclear
- otherwise prioritize one concrete high-value flow
- tie recommendation to Sarah's brand and human-in-the-loop preference

## What to avoid
- generic chatbot-marketing advice
- over-automation of Sarah's DMs
- skipping handoff design
- pretending docs do not exist
