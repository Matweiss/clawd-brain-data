# HEARTBEAT.md - Luke's Daily Operations

## Morning Pulse (7:00 AM PT weekdays)

1. **Check Google Calendar** for today's meetings
   - Use: `gog calendar events primary --from $(date -I)T00:00:00 --to $(date -I)T23:59:59`
   
2. **Pull Granola notes** from yesterday's meetings
   - Check for new transcripts
   - Extract action items
   - Sync to Notion

3. **Review Pipeline** in Google Sheets
   - Flag deals in same stage >7 days
   - Identify deals with no activity >3 days
   - Check for upcoming close dates

4. **Send Mat briefing** via Telegram
   - Today's meetings with prep status
   - Pipeline alerts (if any)
   - Follow-ups due today

## Pre-Meeting Workflow (1 hour before each meeting)

1. **Research prospect**
   - LinkedIn (founders, decision-makers)
   - Recent news/funding
   - Competitive landscape
   - Previous touchpoints

2. **Pull battle cards** from Notion
   - Industry-specific talking points
   - Lucra value props for this vertical
   - Case studies/references

3. **Prepare brief** for Mat
   - 3-5 key talking points
   - Relevant background
   - Suggested agenda

## Post-Meeting Workflow (within 2 hours)

1. **Review Granola transcript**
   - Key decisions made
   - Action items
   - Next steps discussed

2. **Update Pipeline**
   - Move deal stage if applicable
   - Update close date
   - Log last contact

3. **Create follow-up tasks** in Paperclip
   - Set due dates
   - Assign to appropriate agent (or self)

4. **Log notes** to Notion
   - Link to Granola transcript
   - Summary of discussion
   - Open questions

## End-of-Day (6:00 PM PT)

1. Review tomorrow's calendar
2. Set prep reminders for morning meetings
3. Check for urgent follow-ups
4. Update daily metrics

## Weekly (Fridays 4:00 PM PT)

1. Pipeline review
   - Stage movement analysis
   - Forecast accuracy
   - Identify at-risk deals

2. Battle card updates
   - Add new insights from calls
   - Refresh competitive intel

3. Send weekly summary to Mat
   - Deals moved this week
   - Meetings held
   - Pipeline health metrics
